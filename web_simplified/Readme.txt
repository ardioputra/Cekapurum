Buka CMD atau powershell, lalu buat/masuk ke folder project
mkdir project
cd project

inisialisasi node js
npm init

install library
npm install antares-http --save
npm install express

Buat file .html dan .css sesuai frontend
Pastikan ada request ke Node JS
Contoh code ada di bawah

node .\scripts.js
